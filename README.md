# webserv_tester

### Korean

이 테스터는 42의 <i>webserv</i> 서브젝트를 위한 테스터입니다.

다음의 기준으로 평가를 합니다.

1. RFC를 지켰는가
2. 입력 값을 올바르게 검증하는가

이 테스터는 완벽하지 않을 수 있습니다. 정당한 이유가 있다면
테스터가 예상한 것과 다른 결과가 나와도 괜찮습니다.
